THE CODE :

<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /

  # Serve existing files or directories
  RewriteCond %{REQUEST_FILENAME} -f [OR]
  RewriteCond %{REQUEST_FILENAME} -d
  RewriteRule ^ - [L]

  # Redirect everything else to index.html
  RewriteRule ^ index.html [L]
</IfModule>
